# Изменения в fixed-версии

- Добавлено поле «Отапливаемая площадь, м²» при создании пользователя.
- В таблице пользователей можно быстро исправлять и сохранять отапливаемую площадь.
- В экспорте пользователей добавлена отапливаемая площадь.
- В оплатах добавено отображение типа начисления: вода, отопление, другое.
- В оплатах добавено отображение провайдера платежа при наличии.

Проверка выполнена:

```bash
npm ci
npm run build
```
