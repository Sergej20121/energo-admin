Запуск админки в production

1. Скопируйте .env.production.example в .env.production
2. Укажите VITE_API_BASE_URL
3. Выполните npm install и npm run build
4. Разместите папку dist на вашем домене админки

## Изменения в этой сборке
- экспорт отчётов переведён на CSV, чтобы убрать зависимость от небезопасной xlsx-библиотеки
- перед релизом достаточно выполнить `npm install` заново, так как lock-файл удалён намеренно
